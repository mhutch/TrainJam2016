# TrainJam 2016

## Overview

### Theme

Maximum Capacity

### Concept

* Top down vehicle driving game
* Infinite terrain with obstacles
* Pickups that stack up on top of the vehicle and make it more diffcult to drive
* Aim is to reach maximum capacity
