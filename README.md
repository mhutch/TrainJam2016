# Heighten

## Overview

Heighten is a TrainJam 2016 Game. The theme is 'Maximum Capacity'.

### Concept

* Vehicle driving game
* Pickups that stack up on top of the vehicle but get more and more unvieldy
* Aim is to reach the maximum capacity, ten
